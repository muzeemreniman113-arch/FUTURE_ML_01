AI-Powered Sales Forecasting Dashboard

Project Overview
- This project focuses on analyzing historical sales data and predicting future sales trends using machine learning techniques. The goal is to support business decision-making through data-driven insights and visualizations.


Tools & Technologies Used
1.Python (Pandas, Prophet, Matplotlib)
2.Google Colab
3.Power BI
4.GitHub


Dataset
1.Retail sales dataset sourced from Kaggle
2.Data was cleaned and aggregated at a monthly level to reduce noise and capture clear trends and seasonality.


Project Workflow
1.Imported and cleaned the sales dataset
2.Converted date columns and handled missing values
3.Aggregated sales data on a monthly basis
4.Trained a time-series forecasting model (Prophet)
5.Generated future sales predictions with confidence intervals
6.Exported results to CSV
7.Built an interactive Power BI dashboard


Business Insights
1.Monthly sales show an overall upward trend, indicating steady business growth.
2.Seasonal patterns are observed, with certain months contributing higher sales.
3.Sales dips in some months suggest low-demand periods.
4.Forecasted sales indicate continued growth in upcoming months.
5.Confidence intervals highlight uncertainty in predictions, supporting risk-aware planning.
